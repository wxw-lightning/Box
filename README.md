关卡编辑器：Tools-Sokoban-Level Editor
配置关卡：Create-Sokoban-Level Database, 将关卡拖入database, 再将database拖入GameManager中的Database
编辑器操作：可将网格地图看成画布，鼠标就是画笔，选择不同的道具即可涂鸦地图，快速生成对应的机关。也有撤销按钮，撤回上一步操作。

在传统推箱子基础上增加了属性机制，箱子首次到达目标点会触发不同的效果，这样对玩家推箱子的先后顺序，推进去的方位都需要考量。
灵感来源于鸣潮属性，以下是各属性机制和设计思路：
湮灭：箱子被锁定成墙，不可再移动。因为湮灭箱一旦进入目标便会成为障碍，主要考验不同箱子间推动的先后顺序。后续可扩展方向：在合适的实际成为墙，阻挡其他箱子被各种强制位移推到不想要的位置
气动：将四周箱子推开一格。代替玩家在难以抵达的位置触发一次推动。后续扩展方向：当前这个箱子的作用不太大，后续可增加其他游戏中的地形改变，限制玩家行动，气动箱得以发挥。
衍射：将推入方向的一整排箱子推动一格。可以穿透墙壁将箱子推离死角，对箱子被推入目标点的方向有考量。后续扩展方向：一般作为最终或者关键的一次aoe位移，复杂关卡需要先将其他箱子推到合适位置然后统一触发衍射箱
热熔/冷凝：仅有个思路未在此次demo实验：配合地形水池，水池就像墙，玩家和箱子不可进入。但会被冷凝箱子触发冻结，水面变为冰面变得光滑，玩家和箱子都会直接滑到尽头。许多推箱子类游戏都有这样的机制，可参考市面上的扩展。热熔：融化冰面，将其变回为水面：更多是起到阻碍的作用提高难度。
